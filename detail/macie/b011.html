トップページ／MacIE目次／バグ011

hr要素でwidthプロパティの指定値が正しく反映されない(5.x)

症状

hr要素にwidthプロパティを指定したとき、背景やボーダーが指定された幅を超えて表示されることがある。

例示

<style type="text/css">
hr#h1 {
  box-sizing: border-box;
  height: 10px;
  color: #888; /* IE用 */
  background-color: #dcdcdc; /* Moz/Opera用 */
  width: 75%;
}
</style>

<hr id="h1">

幅75%の水平線を表示させています。

スクリーンショット

WinIE6.0での表示（標準モード）

修正状況

MacIE5.1〜MacIE5.2でこの不具合が発生するそうです。
