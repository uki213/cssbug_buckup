トップページ／NN4.x目次／バグ022

画像アンカーのボーダーを消去できない

症状

アンカー内にimg要素を含めると画像の周りにボーダーが表示されるが、これをボーダーのプロパティを用いて無くすことができない。

例示

<style type="text/css">
.sample1 {
  border: 0 none;
}
</style>

<p><a href="b022.html">
<img src="../img/img01.png" alt="★" class="sample1">
</a></p>

borderプロパティを用いてimg要素のボーダーを表示させないようにしています。

スクリーンショット

N7.1での表示

NN4.8での表示

補足

このバグを回避したい場合は、img要素に対しcolorプロパティで親要素の背景色と同じ色を指定してください。ボーダーが背景に溶け込みます。

<style type="text/css">
.sample2 {
  color: #f5f5f5;
  border: 0;
}
</style>

<p><a href="b022.html">
<img src="../img/img01.png" alt="★" class="sample2">
</a></p>

修正状況

NN4.8で不具合の発生を確認しました。
