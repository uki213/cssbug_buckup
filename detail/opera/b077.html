bottom:0;に配置したボックスの下部が閲覧領域の外にはみ出す(7.x)

  • 2004-02-27修正。例示を変更し、再現できないのでスクリーンショットを削除。
  • 2004-03-07修正。例示を変更し、スクリーンショットを追加。

症状

bottom:0; 指定で配置したボックスの下側が閲覧領域の外にはみ出すことがある。

例示

<style type="text/css">
body {
  position: relative;
  left: 0;
  top: 0;
  z-index: 0;
}
</style>
<script type="text/javascript" src="b077.js"></script>

<div style="position:fixed; bottom:0; left:0; width:6em; background:aqua;">
折り返される長い文字列</div>
折り返される長い文字列

履歴の行き来（［戻る］→［進む］）などをすると、固定配置したボックスの下半分が閲覧領域の外にはみ出すことがあります。

スクリーンショット

Opera7.23標準モード

修正状況

  • Opera7.23で不具合の発生を確認しました。
