DOMでiframe要素を生成しようとするとobject要素が生成される(IE5)

症状

DOMのcreateElementメソッドを用いてiframe要素を生成しようとすると、object要素が生成されてしまう。

例示

<script type="text/javascript">
function create() {
    var obj = document.createElement('iframe');
    alert(obj.nodeName);
}
</script>

<a href="javascript:create()">iframe要素生成</a>
iframe要素生成

createElementメソッドでiframe要素を生成し、その要素オブジェクトのnodeNameプロパティをアラートで表示します。

補足

例示はIE5での動作を確認していません。

修正状況

WinIE6.0ではnodeNameプロパティが「iframe」でした。

報告書き込み

【Win-IE5】
document.createElementメソッドでiframe要素を生成しようとすると、
代わりにobject要素が生成されてしまう。
