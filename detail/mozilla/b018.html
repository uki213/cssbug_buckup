フロート配置要素の子要素を絶対配置にすると親要素を無視した配置になる(N6.1)

症状

位置指定された要素をフロートにした場合、その子要素を絶対配置にしても親要素を無視し、初期コンテナブロックを基準にした配置になる。

例示

<style type="text/css">
div#parent {
    float: left;
    position: relative;
    width: 100px;
    height: 100px;
    background:aqua;
}
div#child {
    position: absolute;
    left:30px;
    top:20px;
    width:50px;
    height:50px;
    background:teal;
}
</style>
<div id="parent">親
<div id="child">子</div>
</div>
親
子

親要素（背景が空色）はフロート、子要素（背景が青緑色）は絶対配置です。

スクリーンショット

症状欄、例示欄の一部などを略した状態です。

修正状況
