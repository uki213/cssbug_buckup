終了タグを省略したp要素へのマージン指定が無視される(Moz1.0RC3)

症状

p要素の終了タグを省略しているときに、p要素に指定したマージンが無視されることがある。

例示

<style type="text/css">
div.header {
  background:red;
  padding:0px;
  margin:0px;
}
.body {
  padding:0px;
  margin:0px;
  background:orange;
}
div.footer {
  padding:0px;
  margin:0px;
  background:olive;
}
</style>

<div class="header">↓隙間が出来ます。</div>
<div class="body"><p class="body">↑隙間がイヤです↓</div>
<div class="footer">↑隙間が出来ます。</div>
↓隙間が出来ます。

↑隙間がイヤです↓

↑隙間が出来ます。

修正状況

発生条件がわかりません。

報告書き込み

Mozilla 1.0RC3 Win32
