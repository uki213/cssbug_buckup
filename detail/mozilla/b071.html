特定のスクリプトを実行するとhr要素が点滅する(N7.1)

症状

hr要素や、hr要素を子孫に含む要素のスタイル関連プロパティの値を変更するスクリプトを繰り返し実行すると、hr要素の水平線が小刻みに点滅することがある。

例示

<script type="text/javascript">
var DOM=document.getElementById;
function always() {
  if (DOM) {document.getElementsByTagName('body')[0].style.marginBottom='5em';};
}
if(DOM) {setInterval("always()",100);}
</script>

<hr>

ページのスクロールや文字列の範囲選択を行うと水平線が点滅を始めます。

修正状況

  • Netscape7.1の互換モードで不具合の発生を確認しました。
  • Netscape7.1の標準モード、Mozilla1.6の標準モードと互換モードでこの不具合は発生しないようです。
