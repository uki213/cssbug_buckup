未知の擬似要素／クラスを指定したセレクタは不正(M1.5)

症状

未知の擬似要素や擬似クラスを指定したセレクタは不正なものとして扱うため、これを含む宣言ブロック全体が無視される。

例示

<style type="text/css">
span:invalid , var {
  color: blue;
}
</style>

<ul>
<li><span>span要素</span></li>
<li><var>var要素</var></li>
</ul>
  • span要素
  • var要素

現在、invalidという名称の擬似要素や擬似クラスは定義されていません。

スクリーンショット

Mozilla1.7alpha標準モード

Netscape7.1標準モード

補足

Mozilla.org Bugzillaでは以下の項目が該当します。

  • Bugzilla Bug 145968: CSS parser accepts unknown pseudo-classes/elements

修正状況

  • Mozilla1.5以降の仕様です。
