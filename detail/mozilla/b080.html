visibility:hidden;を指定した要素に:hoverが効かない

症状

visibility:hidden; を指定して非表示にした要素には、:hover擬似クラスの指定内容が効かない。

例示

<style type="text/css">
a:hover {
  visibility: visible;
}
</style>

→Mozillaバグリスト←

2つの矢印に挟まれた部分に非表示のリンクがあります。リンクにマウスポインタなどを載せたときにa要素の文字列が現れることが望まれます。

スクリーンショット

Netscape7.1標準モード

補足

mozilla.org Bugzillaでは以下の項目が該当します。

  • Bugzilla Bug 33736 [ESM/CSS]:hover no longer matches elements with visibility:hidden（mozilla.org Bugzilla）

修正状況

  • Netscape7.1、Mozilla1.7betaの標準モードと互換モードで不具合の発生を確認しました。
